Command Line Interface¶

gst¶

gst: A CLI tool to manage your GitHub Gists.

gst [OPTIONS] COMMAND [ARGS]...

Options

--version¶

Show the version and exit.

auth¶

Authenticate with GitHub by providing a personal access token.

gst auth [OPTIONS]

clone¶

Clone a Gist.

gst clone [OPTIONS] GIST [PATH]

Arguments

GIST¶

Required argument

PATH¶

Optional argument

create¶

Create a new Gist from a file or directory.

gst create [OPTIONS] PATH

Options

-d, --description <description>¶

Description of the Gist.

-w, --web¶

Open the Gist in a web browser after creation.

-p, --public¶

Make the Gist public.

Arguments

PATH¶

Required argument

logout¶

Logout by removing the stored personal access token.

gst logout [OPTIONS]
